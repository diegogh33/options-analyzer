# Options Analyzer · IBKR

Herramienta personal de análisis de opciones financieras con integración Notion.

## Uso

Accede directamente desde cualquier dispositivo:
**https://diegogh33.github.io/options-analyzer**

## Funcionalidades

- Análisis rápido de opciones (Long/Short Call/Put)
- Informe institucional de 10 secciones con precio y earnings en tiempo real
- Histórico de análisis sincronizado con Notion
- Comparación de operaciones con recomendación de Claude
- Compatible con móvil y escritorio

## Tecnología

HTML/CSS/JS · Claude API · Notion MCP · GitHub Pages
